Projeto Rosa Parks

Se trata de uma aplicação web para gestão de Cursinhos Populares (Pré-Vestibulares) com a proposta de se ter um melhor acompanhamento do fluxo de alunos durante os períodos de atividade do(s) cursinho(s).

Nele é possível se realizar o cadastro dos alunos e a ativação ou a inativação dos mesmos (caso o aluno deixe de acompanhar as atividades da organização) com persistência de dados.
